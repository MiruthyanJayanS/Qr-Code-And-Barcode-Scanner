# QR Code and Barcode Scanner

A complete Android application built in **Java**, using **Android Studio**, that scans and decodes both **QR codes** and **barcodes**.

## 📌 About

This is a fully functional Android app designed to scan QR codes and barcodes using the device's camera and instantly decode their contents. Built natively in Java within Android Studio, it's structured as a complete, standalone mobile application rather than a script or library.

## 🛠️ Tech Stack

| Category | Tools |
|---|---|
| Language | Java |
| Platform | Android |
| IDE | Android Studio |

## ✨ Features

- Scans both **QR codes** and **barcodes** using the device camera
- Real-time decoding and display of scanned content
- Built as a complete, installable Android app

## 🚀 Getting Started

### Prerequisites
- [Android Studio](https://developer.android.com/studio) installed
- An Android device or emulator (with camera access for testing on a physical device)

### Setup
```bash
git clone https://github.com/MiruthyanJayanS/Qr-Code-And-Barcode-Scanner.git
```

1. Open the project in **Android Studio**
2. Let Gradle sync and resolve dependencies
3. Connect an Android device (or use an emulator) and click **Run**

### Permissions
This app requires **camera access** to scan codes — make sure camera permissions are granted when prompted on first launch.
